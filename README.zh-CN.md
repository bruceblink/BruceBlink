## 🧠 关于我

- 💻 以后端开发为主，拥有扎实的 **Java** 工程经验
- 🦀 长期使用 **Rust**，关注 ownership / lifetime / async / 并发模型
- 🏗️ 技术兴趣集中在 **系统设计、分布式系统、网络与底层原理**
- 🕷️ 将「爬虫」视为 **数据采集子系统** 而非脚本：
  - 多源数据抓取与异构数据整合
  - 任务调度、失败重试与稳定性设计
  - 数据清洗、归一化与持久化
  - 作为后端服务或工具链的一部分进行集成
- 🚀 偏好构建 **长期可维护的工程型项目**（CLI / Library / GUI 共享同一核心）
- 📚 相信：**理解系统本身，比熟悉框架更重要**
- 🌍 开源爱好者，同时从事技术翻译与文档工作
